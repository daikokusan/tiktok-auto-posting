About This App

🤖

AI Content Generation

Automatically generates short-form vertical video scripts using local LLM (Ollama) based on trending topics in productivity and side-income.

🎬

Automated Video Production

Creates 1080x1920 slide-style videos with narration using FFmpeg, Pillow, and macOS TTS (Kyoko voice).

📅

Scheduled Publishing

Posts to TikTok 3 times daily (7:00, 12:00, 21:00 JST) via the TikTok Content Posting API v2.

✅

Human Approval Step

Each generated video goes through a human review and approval step before it is published to TikTok.

How It Works

1

Topic Generation

The local LLM selects a topic about productivity, side-income, or AI tools relevant to Japanese audiences aged 20-40.

2

Script & Slide Creation

Generates a 5-slide script with narration text, color scheme, and call-to-action automatically.

3

Video Rendering

FFmpeg composes each slide with animated text and synthesized voiceover into a vertical MP4 (1080x1920).

4

Human Review

The creator reviews and approves the generated video before it goes live on TikTok.

5

TikTok Publishing

The approved video is uploaded and published via the TikTok Content Posting API v2 using OAuth 2.0 (PKCE) authentication.

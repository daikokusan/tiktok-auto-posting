2. Description of the App

TikTok Auto Posting is an AI-powered content automation tool that helps content creators automatically generate and publish short-form videos to TikTok. The App uses the TikTok Content Posting API v2 to upload and publish videos on behalf of the authenticated user.

  • Generates video scripts and content using a local AI model (Ollama)
  • Renders vertical short-form videos (1080x1920) using FFmpeg and macOS TTS
  • Requires explicit human approval before each video is published
  • Publishes approved videos to TikTok via the Content Posting API v2
